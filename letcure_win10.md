# 中文 Windows 10 操作系统核心知识点讲义

---

## 一、系统基础与用户界面

### 1. 系统类型与特性
- **Windows 10 是一个多用户、多任务操作系统**（支持多用户登录和多程序同时运行）。
- 窗口特性：
  - 屏幕上可出现多个窗口，但**至多只有一个是活动窗口**。
  - 窗口最小化后，应用程序**被转入后台执行**，并未关闭或暂停。
  - 窗口最大化时，双击标题栏相当于单击**还原按钮**。
  - 将窗口拖动到屏幕顶端，窗口会**最大化**。
  - 对话框：仅可以移动，**不能改变大小**。

### 2. 桌面与任务栏
- **桌面**：整个显示器屏幕，是用户操作的主要界面。
- **任务栏**：最左侧的按钮是**“开始”按钮**，用于打开开始菜单。
- **桌面图标**：实质上是**快捷方式**，删除图标仅删除快捷方式，不会删除程序本身。

### 3. 鼠标操作
| 操作 | 定义 |
| :--- | :--- |
| 单击 | 快速按下并释放鼠标左键 |
| 双击 | 连续两次快速按下并释放鼠标左键 |
| 拖曳 | 按住鼠标左键同时移动鼠标 |
| 右键单击 | 快速按下并释放鼠标右键，弹出快捷菜单 |

## 二、文件与文件夹管理

### 1. 文件系统
- **主流文件系统**：NTFS 取代了 FAT，成为 Windows 的主要文件系统。
- **文件存储单位**：在磁盘中存储文件以**簇**为基本单位。
- **文件路径**：描述文件在磁盘上的目录位置。
- **回收站**：
  - 可以恢复从硬盘中删除的文件或文件夹。
  - 被放入回收站的文件仍然占用**硬盘空间**。
  - `Shift + Delete`：文件不放入回收站而直接删除。

### 2. 文件与文件夹操作
- **新建文件夹**：在资源管理器右窗格空白区域**单击鼠标右键**，在弹出菜单中选择“新建→文件夹”。
- **文件移动/复制**：
  - 直接拖动：在同一磁盘内移动文件。
  - `Shift + 拖动`：在不同磁盘间移动文件。
  - `Ctrl + 拖动`：复制文件到目标位置。
- **文件查找（搜索）**：
  - 根据文件名查找时，至少需要输入文件名的一部分或通配符。
  - 被设置为隐藏的文件，在文件夹选项设置为“显示隐藏文件”时才会被找出。

### 3. 文件名规则
- 可以包含汉字和空格。
- 名称长度一般不超过 256 个字符。
- 不允许出现 `?`、`\`、`<`、`>` 等特殊字符。

## 三、快捷键与系统操作

### 1. 常用快捷键
| 功能 | 快捷键 |
| :--- | :--- |
| 复制 | `Ctrl + C` |
| 剪切 | `Ctrl + X` |
| 粘贴 | `Ctrl + V` |
| 全选 | `Ctrl + A` |
| 全角/半角切换 | `Shift + Space` |
| 输入法切换 | `Ctrl + Shift` |
| 中英文切换 | `Ctrl + Space` |
| 打开开始菜单 | 按下 `<Windows徽标>` 键 或 `Ctrl + Esc` |
| 获得联机帮助 | `F1` |
| 复制全屏幕画面到剪贴板 | `PrintScreen` |
| 复制活动窗口画面到剪贴板 | `Alt + PrintScreen` |

### 2. 键盘输入状态
- 当键盘处于英文大写状态时，按 `Shift + A` 键，输入的是 `a`。

## 四、系统功能与附件程序

### 1. 库功能
- 库中可添加硬盘上的任意文件夹。
- 库中文件夹里的文件**保存在原来的地方**，库中添加的是指向文件和文件夹的快捷方式。

### 2. 附件程序
- **记事本**：用于编辑**纯文本文件**（`.txt`）。
- **画图**：
  - 使用矩形工具画正方形时，需同时按下 `Shift` 键。
  - 画水平、垂直或沿45度方向的直线时，要按 `Shift` 键。

### 3. 系统启动与默认位置
- 启动 Windows 10 后，系统默认的文档保存位置是**我的文档**。
- 安装程序时，程序被安装在**硬盘**中。

## 五、硬件与系统设置

### 1. 硬件要求
- 在 Windows 10 中播放音频文件，必须具有的硬件是**声卡**。
- 对磁盘进行**格式化**时，将在磁盘上创建文件系统。

### 2. 硬盘物理结构
- 由盘片（盘面）、**磁头**、磁道、扇区和柱面组成。

### 3. 显示设置
- 当程序运行界面显示不完整时，正确的做法是**更改系统显示属性，重新设置分辨率**。

## 六、Windows 10 版本与其他概念

### 1. 系统版本
- Windows 10 包含多个版本，其中**移动版**面向尺寸较小、配置触控屏的移动设备。

### 2. 休眠模式
- 在 Windows 的休眠模式下，系统的状态是**保存在硬盘中**的。

### 3. 其他操作系统
- **Unix** 不是微软公司开发的操作系统。
- **DOS** 不是多用户、多任务操作系统。

## 七、文件类型判断

根据文件扩展名判断文件类型：
- **图像文件**：`.bmp`（如 `文章.bmp`）
- 文本文件：`.txt`（如 `照片.txt`）
- 文档文件：`.doc`（如 `歌词.doc`）
- 音频文件：`.wav`（如 `歌声.wav`）

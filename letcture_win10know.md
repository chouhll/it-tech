# Windows 10 核心知识点速记清单

---

## 一、系统基础与界面
1.  **系统类型**：Windows 10 是**多用户、多任务**操作系统。
2.  **窗口规则**：
    -   屏幕可出现多个窗口，但**至多只有一个活动窗口**。
    -   窗口最小化 → 程序转入**后台执行**。
    -   最大化时双击标题栏 → 相当于单击**还原按钮**。
    -   拖窗口到屏幕顶端 → 窗口**最大化**。
    -   对话框：可移动，**不能改变大小**。
3.  **桌面与任务栏**：
    -   桌面 = 整个显示器屏幕。
    -   任务栏最左侧是**“开始”按钮**。
    -   桌面图标本质是**快捷方式**，删除图标≠删除程序。
4.  **鼠标操作**：
    -   单击：快速按放左键。
    -   双击：连续两次快速按放左键。
    -   拖曳：按住左键移动鼠标。
    -   右键单击：弹出**快捷菜单**。

## 二、文件与文件夹管理
1.  **文件系统**：
    -   主流文件系统：**NTFS**（取代 FAT）。
    -   存储基本单位：**簇**。
    -   文件路径：描述文件在磁盘上的**目录位置**。
2.  **回收站**：
    -   可恢复从**硬盘**删除的文件/文件夹。
    -   回收站文件仍占用**硬盘空间**。
    -   `Shift + Delete`：直接删除，不进回收站。
3.  **文件操作**：
    -   新建文件夹：右键空白区 → “新建→文件夹”。
    -   移动/复制：
        -   同盘拖动：移动。
        -   `Shift + 拖动`：跨盘移动。
        -   `Ctrl + 拖动`：复制。
    -   搜索：按文件名查找，可输入部分名称或通配符；隐藏文件需设置“显示隐藏文件”才会被找到。
4.  **文件名规则**：
    -   可含汉字、空格。
    -   长度一般≤256字符。
    -   不可含 `?`、`\`、`<`、`>` 等特殊字符。

## 四、系统功能与附件
1.  **库功能**：
    -   可添加任意文件夹。
    -   库中文件仍在原位置，库只是**快捷方式集合**。
2.  **附件程序**：
    -   记事本：编辑**纯文本文件**（`.txt`）。
    -   画图：
        -   画正方形：矩形工具 + `Shift` 键。
        -   画水平/垂直/45°线：按 `Shift` 键。
3.  **默认位置**：
    -   系统默认文档保存位置：**我的文档**。
    -   程序安装位置：**硬盘**。

---

## 五、硬件与设置
1.  **硬件要求**：
    -   播放音频必须有**声卡**。
    -   磁盘格式化时创建文件系统。
2.  **硬盘结构**：盘片 → **磁头** → 磁道 → 扇区 → 柱面。
3.  **显示设置**：界面显示不完整 → 调整**分辨率**。
4.  **休眠模式**：系统状态保存在**硬盘**中。

---

## 六、版本与其他系统
1.  **Windows 10 版本**：移动版 → 面向小尺寸触控移动设备。
2.  **其他操作系统**：
    -   非微软开发：**Unix**。
    -   非多用户多任务：**DOS**。
